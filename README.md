# laundryserv
This is web page created for the Laundry services webpage

1) It has the two sections those are one is Navigation bar consisting of logo title, navigation links and a user name button and second one is Hero section consisting of motto of the company and one image in the right side.
2) For the navigation section, i used the nav tag in that there are three sections each one for the logo text, navigation links and button respectively, added the hover effect for the navigation links and button, and used display: inline-block to set them in the same line.
3) For the hero section, i used main tag and in that main element i have created two divisions one for the left side content and another for the right side content, given the width of 45vw to equally split the screen for them, and used the same display: inline-block to align them.
